Drop your background files here.

Supported formats:
- .webp
- .png
- .jpg
- .jpeg

How matching works:
- Stage uses "background": "beach_day" in lesson JSON
- Builder looks for a file with the same base name, e.g.:
  - beach_day.webp
  - beach_day.png
  - beach_day.jpg
  - beach_day.jpeg

Recommended:
- Use 16:9 images
- Prefer .webp for smaller output size
